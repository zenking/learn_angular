分页器指令配置

以下配置项可以通过给uib-pagination指令添加属性的方式配置，或是通过uibPaginationConfig 进行全局配置。

  • ng-change : 可以配合ng-model使用，指定一个函数，当前页改变时(ng-model改变)触发。
  • ng-model : 当前页，第1页的值就为数字1。
  • ng-disabled : 设置这个属性来选择是否禁用分页。
  • total-items : 所有页的总条目数。
  • items-per-page(默认值： 10) : 每一页的条目数。
  • max-size(默认值：null) : 显示分页按钮的数量。
  • num-pages只读属性(默认值：angular.noop) : 总页数。
  • rotate(默认值: true) : 是否将当前页始终显示在分页按钮的中间。
  • direction-links(默认值：true) : 是否显示“下一页”和“上一页”按钮。
  • previous-text(默认值：'Previous') : “上一页”按钮的文本。
  • next-text(默认值：'Next') : “下一页”按钮的文本。
  • boundary-links(默认值：false) : 是否显示“第一页”和“最后一页”按钮
  • first-text(默认值：'First') : “第一页”按钮的文本。
  • last-text(默认值：'Last') : "最后一页"按钮的文本。
  • template-url(默认值：'template/pagination/pagination.html') : 指定自定义模版的路径。

页面分页指令配置

可以通过给uib-pager指令添加属性的方式配置，或者通过uibPagerConfig 进行全局配置。

ng-model, total-items, items-per-page 和 num-pages 配置可以参见分页器指令配置。其他的配置如下：

  • align(默认值：true) : 是否将下一页，上一页按钮放在页面两端显示，如果设置为false,则按钮会在中间显示。
  • previous-text(默认值：'« Previous') : ”上一页“按钮文本。
  • next-text(默认值：'Next »') : ”下一页“按钮文本。
  • template-url(默认值：'template/pagination/pager.html') : 指定自定义模版的路径。
  • ng-disabled : 是否禁用分页。
